A modern RTL WordPress theme for Urdu and Arabic content with beautiful Nastaleeq typography.

== Description ==

Qalamik is a professionally designed RTL (Right-to-Left) WordPress theme specifically crafted for Urdu and Arabic content creators. It features:

* Beautiful Nastaleeq typography for authentic Urdu text rendering
* Full RTL language support
* Responsive design (mobile, tablet, desktop)
* Custom Urdu/Arabic font options
* Integrated Urdu keyboard (OpenPad)
* Two customizable sidebars
* Custom logo support
* Custom header and background
* Translation ready
* SEO optimized
* Block editor compatible
* Threaded comments
* Social media integration
* Widget areas

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support RTL languages? =

Yes! Qalamik is specifically designed for RTL languages like Urdu and Arabic.

= Can I use custom fonts? =

Yes, the theme includes options for multiple Urdu and Arabic fonts including Jameel Noori Nastaleeq, Noto Nastaliq Urdu, Mehr Nastaliq, Amiri, and Cairo.

= Is this theme responsive? =

Yes, Qalamik is fully responsive and works on all devices.

= Can I customize the colors? =

Yes, you can customize colors through the WordPress Customizer.
